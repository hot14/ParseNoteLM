## 리스크 및 대응책

### 기술적 리스크
- **OpenAI API 제한**: 대안 모델 준비 (Anthropic Claude)
- **벡터 검색 성능**: 인덱스 최적화 및 쿼리 튜닝
- **파일 처리 오류**: robust한 에러 핸들링 구현

### 일정 리스크
- **개발 지연**: 핵심 기능 우선 구현, 부가 기능은 연기
- **통합 이슈**: 주간 단위 통합 테스트 실시

### 사용자 리스크
- **낮은 정확도**: 프롬프트 엔지니어링 개선
- **사용성 문제**: 조기 사용자 테스트 실시
