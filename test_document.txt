ParseNoteLM 사용자 매뉴얼

1. 개요
ParseNoteLM은 AI 기반의 문서 분석 및 질의응답 시스템입니다. 사용자는 PDF나 텍스트 문서를 업로드하고, AI와 자연어로 대화하며 문서의 내용을 쉽게 탐색할 수 있습니다.

2. 주요 기능
- 문서 업로드 및 자동 분석
- RAG(검색 증강 생성) 기반 질의응답
- 팀 협업 및 프로젝트 관리
- 실시간 채팅 인터페이스

3. 사용 방법
가. 회원가입 및 로그인
나. 프로젝트 생성
다. 문서 업로드
라. AI와 질의응답

4. 질의응답 예시
Q: 이 시스템의 주요 기능은 무엇인가요?
A: ParseNoteLM의 주요 기능은 문서 업로드 및 자동 분석, RAG 기반 질의응답, 팀 협업, 실시간 채팅 등입니다.

Q: RAG란 무엇인가요?
A: RAG(Retrieval-Augmented Generation)는 검색 증강 생성 기술로, 문서에서 관련 정보를 검색한 후 이를 바탕으로 정확한 답변을 생성하는 AI 기술입니다.

5. 기술 스택
- 백엔드: FastAPI, SQLAlchemy, OpenAI API
- 프론트엔드: React, TypeScript, Tailwind CSS
- 데이터베이스: SQLite (개발), PostgreSQL (프로덕션)
- 벡터 검색: FAISS

6. 고급 기능
- 문서 청킹 및 임베딩
- 유사도 기반 검색
- 컨텍스트 인식 답변 생성
- 멀티턴 대화 지원
